# Teste Ally Hub
## Começando

Primeiro, para baixar:

```bash
git clone "(Link do repositório)"
```
ou baixe o a pasta zipada 

abra a pasta no seu terminal e insira para instalar todas a dependências:

```bash
npm install
```

para rodar a aplicação:
```bash
npm run dev
```
